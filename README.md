# plato-timing

Tensor MIDI timing for PLATO room agent coordination.

## What It Does

Provides Tensor MIDI-based timing primitives for coordinating actions across PLATO room agents. Handles clock synchronization, event scheduling, and temporal coordination.

## Ecosystem

- **[plato-nervous](https://github.com/SuperInstance/plato-nervous)** — Related (timing may inform signal chain scheduling)
- Used externally by [luciddreamer-ai](https://github.com/SuperInstance/luciddreamer-ai) for podcast improv timing

This crate is standalone within the PLATO ecosystem — no internal PLATO dependencies.
